🎯 Resumo da regra (em uma linha)

  • Pergunta → R1 (aprovador) → resposta executor → R2 (aprovador) → resposta executor → plano de ação (se houver)
  • Após R2 final, se auditor pedir → aprovador responde no card 🟨 destacado em amarelo logo abaixo da nota final do aprovador
  • Auditor vê esse card 🟨 + botão Conforme / NC ENTRE R2 do aprovador e R1 do auditor
  • Se Auditor → NC → cria R1 do auditor (plano de ação para o executor)
  • Se Auditor → Conforme → finaliza, sem nova rodada
